# `plot.tscn` — plantable dirt mound

`Plot` (`Area2D`, group `plot`), script: `scripts/plot.gd`. A passive detectable, same pattern as `seed.tscn` — no `HoverZone` sub-node, the whole area is directly what the player's `WaterSensor` polls for. Children: `CollisionShape2D` (`CircleShape2D`), `Mound`/`MoundShade` (`Polygon2D`s, the dirt visual, always present), `Marker` (a small pulsing glow diamond, visible only while empty, signalling "plantable here"), `PlantPuff` (`CPUParticles2D`, one-shot dust burst) and `PlantSound` (`AudioStreamPlayer2D`, `assets/audio/seed_plant.wav`).

`is_empty` starts `true`. While empty, `_process()` pulses `Marker.modulate.a` via a sine wave so it reads as plantable at a glance without being static. `plant(type: PlantData.PlantType)` (called by `player.gd` when the bee hovers an empty plot while carrying a seed) is a no-op if already occupied; otherwise it flips `is_empty` to `false`, hides `Marker`, plays `PlantPuff`/`PlantSound` (pitch-varied per the fit-and-finish guidance), does a `create_tween()` scale-pop (`Vector2.ZERO → Vector2.ONE`, back-eased, matching `seed.tscn`'s pop-in style), and emits `seed_planted(hybrid_type, at_position)`.

The plot never spawns a `Seedling` itself — per REQUIREMENTS.md, plots/seeds/plants stay passive detectables. `main.gd` connects to every `Plot`'s `seed_planted` the same way it already connects to every `Seedling`'s `seed_popped`, and instantiates `seedling.tscn` at the plot's position with `growth = 0`, `bloom_type` set from the hybrid type — the actual scene spawning lives on `Main`.
